Elementos 

- Pokemon: id, frente, reverso
- Generador de prejas
- Estadisticas de juego: tiempo restante, puntaje 
- Coordinador de la partida
- Volteador de pokemon seleccionados

Responsabilidades

- Crear las parejas de pokemons aleatoriamente
- Llevar la cuenta de cuantas parejas se han encontrado 
- Llevar el control del puntaje 
- Indicar el tiempo restante de la prtida
- Determinar cuando se pierde el juego
- Determinar cuando se gana el juego

Eventos

- Se crea el tablero
- Se muestra el primer pokemon
- Se muestra el segundo pokemon 
- Los dos pokemons coinciden
- Los dos pokemons no coinciden
- Se gana el juego
- Se pierde el juego cuando termina el tiempo

Secuencia


- Se crea el tablero con 25 parejas de pokemones distribuidos aleatoriamente
- Se inicializa el coronometro
- El jugador elige un pokemon al azar y este se voltea
- El jugador elige otro pokemon al azar y este se voltea
- Se comparan ambos pokemones
- Si son iguales
    - Se registra el pokemon encontrado 
    - Se incrementa el puntaje
- Sino
    - Se muestran los dos pokemones por un tiempo
    - Se voltean ambos pokemones para ocultarlos
- El usuario vuelte a elegir dos pokemonones y el proceso se repite hasta que encuentre todas las parejas
